# Discord Updater For Linux

You are a Linux user and always need to install the newest version by hand?
This script will automate this process for you.

# Installation

1. Clone this repository
2. Run `chmod +x ./launcher.sh`
3. Run `./launcher.sh`

> [!NOTE]
>
> If you want to run from the terminal simply put 
> `alias discord='[PATHTOREPOSIOTRY]/launcher.sh'` into your `.bashrc`
